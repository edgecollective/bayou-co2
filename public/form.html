Bayou-CO2

Cooperative CO2 Sensor Data Storage and Analysis

Bayou-CO2 is designed to be a simple way of capturing, storing, and retrieving CO2 sensor data. Its creation was motivated by recent work (e.g. by Jimenez et al.) indicating that monitoring indoor CO2 levels can (in some contexts) help to assess the risk of airborne SARS-CoV-2 transmission.

Free and Open Source. All the code for Bayou-CO2 is free and open source. Feel free to fork it, modify it, improve it; and we'd love it if you ran your own version of it!

Open Data. You can download all historical sensor data as CSV or JSON.

Use any sensor you like! Bayou-CO2 can receive data from any device capable of making HTTP POST requests. Many inexpensive microcontrollers can be programmed to do this (when connected to suitable CO2 sensors). You can find some guidance on doing this here.

Funded through donations. At Pioneer Valley Open Science and Edge Collective, we use donations and pay-as-you-can arrangements to support Bayou-CO2 data hosting and development. To help keep this site running, please visit our site on Open Collective here. Cheers!

PVOS Open Hardware CO2 Sensor Kits. We have developed an Open Hardware CO2 monitoring kit, based on an ESP32 microcontroller and an SCD30 CO2 sensor. The design files, code, and instructions for building are all available; we are also able to distribute pre-assembled kits that are pre-programmed to send CO2 data to Bayou.
